Repo-Backed Evidence App

Ask a question or test a claim against the library.

This app reads a repo-committed library of books and files. A user can ask a question, give a prompt, or state a claim, and the app gathers evidence for and against it before producing a grounded response.

How It Works

  • All shared evidence lives in the repo library
  • The app weighs supporting and opposing passages
  • The final response is written from retrieved evidence only
  • Quoted phrases are optional exact constraints
  • Local mode can add new books into the repo
