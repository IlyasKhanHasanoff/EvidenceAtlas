# Repo PDF Drop

Developers can place shared source PDFs in this folder, commit them, and push them to GitHub.

After the files are in the repo:

1. Run the app locally.
2. Add a subject and optional sub-subject in the upload panel.
3. Use `Import repo-drop PDFs`.

The import keeps the original files in this folder and also copies indexed versions into `docs/library/pdfs/` for the shared library.
